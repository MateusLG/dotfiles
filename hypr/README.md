
# hypr

Configurações do Hyprland.

- `input.conf` — layout de teclado (us + br abnt2, troca com Alt+Espaço)

- `bindings.conf` — atalhos customizados (AltGr+Q/W/E para /, ?, °)

- `monitors.conf` — configuração de monitores: LG (HDMI-A-1) como principal em `0x0`, notebook (eDP-1) à esquerda em `-1366x0`; workspace 1 padrão no LG quando conectado

- `hypridle.conf` — gerenciamento de idle: screensaver aos 30min, lock + backlight do teclado off aos 30min30s, tela off (DPMS) aos 31min
