
# hypr

Configurações do Hyprland.

- `input.conf` — layout de teclado (us + br abnt2, troca com Alt+Espaço)

- `bindings.conf` — atalhos customizados (AltGr+Q/W/E para /, ?, °)

- `monitors.conf` — configuração de monitores: LG (HDMI-A-1) como principal em `0x0`, notebook (eDP-1) à esquerda via `auto-left`; workspace 1 padrão no LG quando conectado

- `hypridle.conf` — gerenciamento de idle: screensaver aos 5min, backlight do teclado off aos 5,5min, tela off (DPMS) aos 10min, bloqueio de sessão aos 15min
