
# scripts

Scripts utilitários.

- `work.sh` — conecta na VPN (FortiClient) e abre uma sessão RDP no Remmina automaticamente. Parametrizado via `VPN_PROFILE`, `VPN_USER` e `RDP_PROFILE`.
- `lgfetch.sh` — wrapper sobre o `fastfetch` que troca o logo e a cor de destaque conforme o tema atual do Omarchy (lido de `~/.config/omarchy/current/theme.name`): bonsai verde para `lg-umbra`, tubarão "Jaws" azul para `lg-abissal`, fastfetch padrão nos demais. Instalado em `~/.local/bin/lgfetch`.
